   - Footer does NOT need SectionObserver (not a nav target)

IMPORTANT: Use native `<a href="#section-id">` for all nav links, NOT Next.js `<Link>`. Per research, Next.js Link does not respect scroll-padding CSS. For a single-page site, native anchors are correct.
