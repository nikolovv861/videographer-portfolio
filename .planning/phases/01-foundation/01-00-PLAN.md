  <files>

  <action>
1. Install Playwright:
   ```bash
   npm init playwright@latest -- --quiet --browser=chromium
   ```
   If the interactive installer causes issues, install manually:
   ```bash
   npm install -D @playwright/test
   npx playwright install chromium
   ```

2. Configure `playwright.config.ts`:
   - baseURL: "http://localhost:3000"
   - testDir: "./tests"
   - testMatch: "**/*.spec.ts"
   - projects: chromium only (for speed)
   - webServer: { command: "npm run dev", url: "http://localhost:3000", reuseExistingServer: true }
   - use: { screenshot: "only-on-failure" }

3. Install Vitest:
   ```bash
   npm install -D vitest
   ```

4. Create `vitest.config.ts`:
   ```typescript
   import { defineConfig } from "vitest/config";
   import path from "path";

   export default defineConfig({
     test: {
       include: ["tests/**/*.test.ts"],
     },
     resolve: {
       alias: {
         "@": path.resolve(__dirname, "."),
       },
     },
   });
   ```

5. Add test scripts to package.json:
   - "test": "vitest run"
   - "test:e2e": "npx playwright test --project=chromium"
   - "test:all": "vitest run && npx playwright test --project=chromium"
  </action>

  </verify>
