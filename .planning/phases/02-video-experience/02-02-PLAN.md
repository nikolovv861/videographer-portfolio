**hooks/useScrollLock.ts** -- Custom hook that locks body scroll when modal is open:

```typescript
export function useScrollLock(isLocked: boolean): void
```

Implementation (per research Pattern 3):
- When `isLocked` becomes true:
  1. Calculate scrollbar width: `window.innerWidth - document.documentElement.clientWidth`
  2. Capture current `window.scrollY`
  3. Set `document.body.style.overflow = 'hidden'`
  4. Set `document.body.style.paddingRight = ${scrollbarWidth}px` (prevent layout shift)
  5. Set `document.body.style.position = 'fixed'` (iOS Safari fix)
  6. Set `document.body.style.top = -${scrollY}px`
  7. Set `document.body.style.width = '100%'`
- Cleanup (when `isLocked` becomes false or component unmounts):
  1. Reset all body styles to empty strings
  2. Restore scroll position: `window.scrollTo(0, scrollY)`
- Use `useEffect` with `[isLocked]` dependency.
- Mark as `"use client"` at top of file.

**components/ui/ProjectCard.tsx** -- Individual project card with hover video preview:

Props: `{ project: Project; onClick: () => void }`

This is a client component (`"use client"`).

Structure:
- Outer `div` with `onMouseEnter` and `onMouseLeave` handlers, `onClick` calls the onClick prop, `cursor-pointer`.
- **Thumbnail area**: `relative aspect-video overflow-hidden rounded-sm` container.
  - `<img>` for thumbnail: `absolute inset-0 h-full w-full object-cover` with `src={project.thumbnailUrl}` and `alt={project.title}`. Use a dark gray placeholder gradient if image fails to load.
  - `<video>` for hover preview: `absolute inset-0 h-full w-full object-cover` with `ref={videoRef}`, `src={project.videoUrl}`, `muted`, `loop`, `playsInline`, `preload="none"`. Opacity transitions: `opacity-0` at rest, `opacity-100` on hover (use state to track hover).
- **Hover effects** (per user decision):
  - At rest: flat, no border/shadow. `transition-all duration-300`.
  - On hover: `scale-[1.02]`, `ring-1 ring-gold/50` (gold border glow), title text shifts to `text-heading` from `text-body`.
  - Apply scale to the outer container via `group` + `group-hover:scale-[1.02]`.
- **Info area**: Below thumbnail, `p-6` padding.
  - Category: `text-sm tracking-wide text-body uppercase`.
  - Title: `mt-2 font-heading text-lg font-semibold text-heading`.

Mouse handlers:
- `handleMouseEnter`: Set hover state true, call `videoRef.current?.play()`.
- `handleMouseLeave`: Set hover state false, pause video, reset `currentTime = 0`.

**Mobile interaction** (per discretion -- tap opens modal directly):
- On touch devices, no hover preview. The `onClick` handler opens the modal directly.
- The hover video preview is a desktop-only enhancement. No special mobile code needed -- touch devices don't trigger mouseEnter.
