  <action>
**VimeoEmbed.tsx** -- Reusable Vimeo iframe wrapper supporting two modes:

Props: `{ vimeoId: string; mode: "background" | "player"; className?: string }`

- `background` mode: Uses URL `https://player.vimeo.com/video/${vimeoId}?background=1&autopause=0`. No visible controls, autoplay muted looped. The iframe should be absolutely positioned with `min-w-full min-h-full` centered via translate to cover the container regardless of aspect ratio. Container is `absolute inset-0 overflow-hidden`.
- `player` mode: Uses URL `https://player.vimeo.com/video/${vimeoId}?color=c9a96e&title=0&byline=0&portrait=0`. Standard Vimeo player with controls. Renders in an `aspect-video` container with responsive width. Allow fullscreen.
- Both modes: `style={{ border: 0 }}`, `allow="autoplay; fullscreen"`, descriptive title attribute.

**VideoPlaceholder.tsx** -- Dev placeholder shown when no real video is configured:

Props: `{ label?: string; className?: string }`

- Dark background (`bg-foreground/5`), centered Play icon from lucide-react (`h-12 w-12 text-foreground/30`).
- Optional label text at bottom (`text-sm text-body`).
- `aspect-video` container with `relative flex items-center justify-center`.
- This is used during development before real Vimeo IDs are configured.

**Parallax effect** (per Claude's discretion recommendation -- content fade+lift):
- Track `scrollY` via `window.addEventListener('scroll', ..., { passive: true })` in a useEffect.
- Calculate `opacity = Math.max(0, 1 - scrollY / 600)` and `translateY = scrollY * 0.3`.
- Apply to the content wrapper via inline `style={{ opacity, transform: translateY(...) }}`.
- Only compute when `scrollY < window.innerHeight` (no work when hero is off-screen).

**Content layout** (matching existing HeroPlaceholder center-aligned pattern):
- `relative z-10 mx-auto max-w-4xl px-6 text-center md:px-8` wrapper centered vertically with `flex items-center justify-center min-h-screen`.
- `h1` with `font-heading text-4xl font-bold tracking-tight text-heading md:text-5xl lg:text-6xl` -- renders `heroContent.headline`.
- `p` with `mt-6 font-heading text-lg tracking-wide text-body md:text-xl` -- renders `heroContent.tagline`.
- Two buttons in a `mt-12 flex flex-col items-center gap-4 sm:flex-row sm:justify-center`:
  - `<Button href={heroContent.ctaWork.href}>{heroContent.ctaWork.label}</Button>`
  - `<Button variant="cta" href={heroContent.ctaContact.href}>{heroContent.ctaContact.label}</Button>`

**Scroll-down indicator** (per discretion -- animated chevron):
- Absolute positioned at bottom center of hero section.
- `ChevronDown` icon from lucide-react with CSS bounce animation.
- Use Tailwind `animate-bounce` class. `text-foreground/40` for subtle appearance.
- Wrap in an anchor `href="#work"` so clicking it scrolls to the work section.
