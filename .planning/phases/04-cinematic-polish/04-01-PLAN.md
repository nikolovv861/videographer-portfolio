  <files>package.json, app/globals.css, components/effects/FilmGrain.tsx, components/effects/GoldDivider.tsx, components/effects/ScrollReveal.tsx</files>
  <action>
1. Run `npm install motion` to add the motion animation library.

2. Create `components/effects/FilmGrain.tsx`:
   - "use client" component
   - Uses SVG feTurbulence filter approach (self-contained, no external assets)
   - Props: `className?: string` for positioning flexibility
   - Renders an absolutely positioned div with pointer-events-none, containing an SVG with:
     - feTurbulence filter: type="fractalNoise", baseFrequency="0.65", numOctaves="3", stitchTiles="stitch"
     - animate element on seed attribute: from="0" to="100" dur="0.5s" repeatCount="indefinite" (creates flickering effect)
     - rect filling 100% width/height applying the filter
   - Opacity: 0.06 (within the 5-8% range specified by user)
   - Export: `FilmGrain`

3. Create `components/effects/GoldDivider.tsx`:
   - "use client" component using motion from "motion/react"
   - Renders a centered container (flex justify-center, py-8) with a motion.div inside
   - The motion.div: h-px, max-w-md, w-full, bg-gold (champagne gold from theme)
   - Animation: initial={{ scaleX: 0 }}, whileInView={{ scaleX: 1 }}, viewport={{ once: true }}, transition={{ duration: 1.2, ease: "easeInOut" }}, style={{ transformOrigin: "center" }}
   - Center-outward expansion mirrors the loading intro's expanding line per user decision
   - Export: `GoldDivider`

4. Create `components/effects/ScrollReveal.tsx`:
   - "use client" component using motion from "motion/react"
   - Two exports:
     a. `ScrollReveal` - Container component with variants: hidden={} / visible={ transition: { staggerChildren: 0.15 } }. Uses initial="hidden", whileInView="visible", viewport={{ once: true, margin: "-100px" }}. Accepts className and children props.
     b. `ScrollRevealItem` - Child wrapper with variants: hidden={ opacity: 0, y: 20 } / visible={ opacity: 1, y: 0, transition: { duration: 0.6, ease: "easeOut" } }. Renders a motion.div with the item variants. Accepts className and children props.
   - The 20px translate-up is within the 20-30px range per user decision ("sections appear rather than fly in")
   - Stagger of 150ms is within the 150-200ms range per user decision

5. Add to `app/globals.css` (append before closing, after the marquee keyframe):
   ```css
   @media (prefers-reduced-motion: reduce) {
     *,
     *::before,
     *::after {
       animation-duration: 0.01ms !important;
       animation-iteration-count: 1 !important;
       transition-duration: 0.01ms !important;
       scroll-behavior: auto !important;
     }
   }
   ```
   This globally disables all CSS animations for reduced-motion users per FX-07.
  </action>
